# Finance Backend API

## Overview
Backend system for managing financial transactions with role-based access control.

## Features
- User authentication (JWT)
- Role-based access (Admin, Analyst, Viewer)
- CRUD operations for transactions
- Dashboard analytics

## Tech Stack
- Node.js
- Express.js
- SQLite
- JWT
